# VITTORIA HOME — инструкция для менеджеров

Как вести клиента, чтобы он в приложении видел актуальный статус заказа. Коротко и по делу.

---

## Главный принцип

Вы работаете **как обычно — в amoCRM**. Приложение клиента само подхватывает изменения. Ваша задача — вовремя двигать сделку по этапам воронки **«VITTORIA HOME»** и заполнять несколько полей.

```
Вы двигаете сделку по этапу в amoCRM ──▶ клиент в приложении сразу видит этот этап и % готовности
```

---

## Шаг 1. Завести клиента в воронку «VITTORIA HOME»

- Когда сделка переходит в производство, переместите её в воронку **«VITTORIA HOME»** (или создавайте такие сделки сразу в ней).
- В сделке должен быть **контакт с телефоном** — по этому телефону клиент входит в приложение. Без телефона приложение не сможет показать заказ.

## Шаг 2. Двигать сделку по этапам (это и есть «статус для клиента»)

Этапы воронки «VITTORIA HOME» — это то, что видит клиент:

1. **Подготовка для производства**
2. **Деталировка**
3. **Поступление материалов на склад**
4. **Производство изделия**
5. **Передача готового изделия на склад**
6. **Проверка комплектности товара**
7. **Готовность к передаче клиенту**

Просто перетаскивайте сделку на нужный этап по мере продвижения заказа — клиент сразу увидит обновление и получит уведомление.

## Шаг 3. Заполнять служебные поля (что ещё видит клиент)

В карточке сделки есть поля для приложения:
- **vittoria_progress** — процент готовности (0–100). Показывается клиенту полосой прогресса.
- **vittoria_admin_comment** — короткий комментарий клиенту (например, «Ваш заказ на финальной проверке»).
- **vittoria_prepayment** — сумма предоплаты (для блока финансов).
- **vittoria_partner_user_id**, **vittoria_partner_services** — для партнёрских услуг (заполняется при необходимости).

> Этап берётся из положения сделки в воронке, а процент/комментарий — из этих полей. Заполняйте их, чтобы клиенту было понятнее.

## Шаг 4. Отвечать клиенту в чате

- Клиент может написать вопрос прямо из приложения.
- Сообщения видны в **веб-панели → раздел «Чат»**: слева список диалогов (с пометкой непрочитанных), справа переписка. Отвечайте текстом — клиент получит уведомление.
- Среднее ожидаемое время ответа в приложении заявлено как «в течение 2 часов» — старайтесь не задерживать.

---

## Что видит клиент в приложении

- Номер договора и изделие.
- Финансы: стоимость, предоплата, остаток.
- Текущий этап + процент готовности.
- Ваш комментарий (если заполнен).
- Кнопку чата и телефон сервиса.

## Частые вопросы

- **Клиент не может войти.** Проверьте, что его телефон указан в контакте сделки (формат `+7XXXXXXXXXX`).
- **Клиент не видит обновления.** Убедитесь, что сделка в воронке «VITTORIA HOME» и переведена на нужный этап.
- **Нужно срочно сообщить клиенту.** Напишите в чат из веб-панели или заполните комментарий в сделке.

---

## Памятка
- Двигаете сделку по этапам «VITTORIA HOME» → клиент видит статус.
- Заполняете процент и комментарий → клиенту понятнее.
- Отвечаете в чате из веб-панели → клиент спокоен.
